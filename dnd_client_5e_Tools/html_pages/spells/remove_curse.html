Remove Curse

\n
\n
\nPHB\np271\n
\n
\n
3rd-level abjuration
Casting Time: 1 action
Range: Touch
Components: V, S
Duration: Instantaneous

At your touch, all curses affecting one creature or object end. If the object is a cursed magic item, its curse remains, but the spell breaks its owner's attunement to the object so it can be removed or discarded.

Classes: Cleric, Paladin, Warlock, Wizard
Subclasses: Arcane Trickster Rogue, Clockwork Soul Sorcerer, Divine Soul Sorcerer, Eldritch Knight Fighter
Subclasses (legacy): Favored Soul v2 (UA) Sorcerer, Favored Soul v3 (UA) Sorcerer
Source: PHB, page 271. Available in the SRD.
